Work Showcase

MedScribe - Doctor-Patient Record Management

"MedScribe" is a comprehensive doctor-patient record management system that I contributed to as a key developer. I designed intuitive UI flows using React.js and implemented backend CRUD operations with Node.js and Sequelize ORM, securely managing patient data, SOAP notes, and appointments in a CloudSQL database. The platform integrates audio consultations converted to text using Google Cloud Speech-to-Text and stored in Google Cloud Storage (GCS). Additional features include a token-based appointment system and role-based access control for multi-role users (Admin, Doctor, Technician, Front Desk).

Twilio IVR Configuration - Interactive Voice Response System

I developed robust IVR systems for restaurant and dental clients using TwiML and Node.js. The IVR system supports speech recognition and DTMF inputs, enabling interactive caller engagement and efficient customer service. This project involved configuring welcome prompts, call routing, and input handling, enhancing client communication channels.
